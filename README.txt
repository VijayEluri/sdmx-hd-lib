=== SDMX-HD Parser ===

This project is a SDMX-HD Parser. It is being developed to aid in the OpenMRS
to TRACnet integration. This parser allows a complex SDMX-HD message to be
parser into a data structure that is easier to work with.

Currently this parser is in development and can only parser the DSD part of
an SDMX-HD message.
